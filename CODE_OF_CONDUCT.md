# Code of Conduct

## Our standards

This project aims to be respectful, constructive, and welcoming.

Examples of expected behavior include:

- being respectful in disagreement and critique
- assuming good intent while still addressing harm clearly
- focusing feedback on ideas, code, and decisions rather than people
- being welcoming to contributors with different backgrounds and experience levels
- respecting privacy, boundaries, and community norms

## Unacceptable behavior

Examples of unacceptable behavior include:

- harassment, discrimination, or hate speech
- personal attacks, insults, intimidation, or threats
- sexualized language or unwanted sexual attention
- doxxing or sharing private information without permission
- deliberate disruption, trolling, or persistent bad-faith engagement
- spam, promotional abuse, or derailing community spaces

## Scope

This code of conduct applies to project spaces under the `NWarila` account, including issues, pull requests, discussions, code review, and other community interactions related to these repositories.

## Reporting

If you experience or witness behavior that violates this code of conduct, report it privately to the maintainer through the [NWarila GitHub profile](https://github.com/NWarila).

Please include links, screenshots, or other context that will help with review. Reports will be handled as promptly and confidentially as practical.

## Enforcement

The maintainer may remove content, lock conversations, reject contributions, or block participants whose behavior violates this code of conduct.

Good-faith reporters will not be penalized for raising concerns.

This code of conduct supplements GitHub's Terms of Service and Acceptable Use Policies.
